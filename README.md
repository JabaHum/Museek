# Loop
A Music Player with Neumorphic design. <br>
Playstore link: https://play.google.com/store/apps/details?id=com.marcqtan.samplemusic <br>

<h3>Technologies used: <br>Motion Layout, Material Design, MVVM, DataBinding, LiveData, Retrofit, RXJava, SoundCloud API, Glide, ViewBinding, MediaSession, Exoplayer</h3>



A collection of NCS top tracks. No ads. No registration. No subscription required. 

<b>Features :</b><br>
◾Neumorphism Design <br>
◾Material Design<br>
◾High quality audio tracks<br>
◾Smooth and simple animation<br>
◾Notification Widget<br>
◾Lock Screen music widget<br>
◾Background playing enabled<br>
◾Tracks are updated frequently<br>

<b>Disclaimer:</b>
All tracks are from NoCopyrightSounds (NCS) and powered by SoundCloud® API with usage terms complied.
